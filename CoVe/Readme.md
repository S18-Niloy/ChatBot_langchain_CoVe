## Getting Started

### Prerequisites

- Python 3.x
- Jupyter Notebook or any compatible environment

### Installation

1. Clone the repository:

```bash
git clone https://github.com/S18-Niloy/ChatBot_langchain_CoVe.git
```
2. Install dependencies:
```
pip install -r requirements.txt
```
```
cd CoVe
```
```
cd src
```
